堆排序 (heap sort)

堆(heap) 是一种完全二叉树，分为两种，大顶堆和小顶堆，满足以下关系:

  • 大顶堆: arr[i] >= arr[2i+1] && arr[i] >= arr[2i+2]
  • 小顶堆: arr[i] <= arr[2i+1] && arr[i] <= arr[2i+2]
    通常升序用大顶堆，降序用小顶堆。
    基本思路:
  • 将无序数组变成大顶堆
  • 将首个数值换到最后一个
  • 将剩下的继续调整成大顶堆
